# Future Roadmap

## Q3 2026: Intelligence Layer
- **AI Resume Screening:** Parse resumes, score against job requirements, flag top candidates
- **Candidate Ranking:** Multi-factor scoring (assessment + resume + referral)
- **Talent Analytics:** Funnel metrics, time-to-hire, source effectiveness dashboards

## Q4 2026: Interview & Proctoring
- **AI Interview Bot:** Async video/text interviews with structured evaluation
- **AI Proctoring:** Tab switch detection, webcam monitoring, plagiarism detection
- **Live Coding Sessions:** Real-time collaborative assessment with interviewer

## Q1 2027: Enterprise Integration
- **ATS Integration:** Greenhouse, Lever, Workday connectors
- **SSO for Admin:** SAML/OIDC enterprise login
- **Multi-tenant:** Separate orgs with isolated data
- **Custom Branding:** White-label per client

## Q2 2027: Scale & Platform

- **Question marketplace:** Shared question banks across teams

- **Pair programming simulation**
